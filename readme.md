### Scope:

The scope of this gulp task is to facilitate the process of changing partials in multiple static pages. Multiple templates and partials are supported.

### Prerequisites:

- latest version of node.js

- ```bash
  npm install -g gulp
  npm install -g gulp-cli
  
  # clone repo
  npm install
  ```

#### Generating the templates or static pages:

Assuming that you have all the prerequisites, you should be able to just run the below command and the static pages will be generated to the **/out** directory.

```bash
gulp render
```
